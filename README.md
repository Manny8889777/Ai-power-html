## 🚀 How to Run

### Option 1 — GitHub Pages (Recommended)
1. Fork this repo
2. Go to Settings → Pages
3. Select `main` branch → Save
4. Open: `https://yourusername.github.io/naseerai`

### Option 2 — Run Locally
```bash
git clone https://github.com/manny8889/naseerai
cd naseerai
# Just open index.html in your browser!
```

### ⚙️ Setup API Key
This app uses the **Anthropic Claude API**.

1. Get a free API key at: https://console.anthropic.com
2. Open `index.html`
3. The app calls the API directly from the browser

> **Note:** For production, use a backend proxy to keep your API key secure.
